Welcome to Enoya-one LTD developer test.

  • Thank you for accepting the option to participate in this test
  • This is a simple HTML with a table and a form, Your task is to transform this HTML form to an interactive form which communicate with a json file. we urge you to read the full task details and then decide how to do it and how to invest your time, remember you have 3 hours to complete.
  • You should do the following:
    • Download the following file http://general.aerocrs.systems/devtest/data.json and place it in your project's folder.
    • This file contains records.
    • Pull the records from the json file, and replace the static html table below with the records from the file.
    • The form should be able to add new records to the file and present it in the table below.
    • Activate the icon which will delete the record from the file.
    • Sort the table by the birth date.
    • Update data of a row using update icon (Clicking the icon should show the record in the form and "Submit" would update it).
    • With every write to the json file, you need to send the inserted data to http://general.aerocrs.systems/devtest/curl.php as POST using CURL.
      Example for POST information to be sent to the URL: email=jonathan@aerocrs.com&firstname=Jonathan&lastname=Rose
      (you will get a OK/BAD response from the CURL, no need to verify it.).
    • The layout of the page, css, date-formats etc. should not be changed
    • The code should be written in PHP, keep it as simple as possible
    • PHP 5.6 and above should be used
    • In the HTML source of this file, there is a syntax error, fix it.
  • You should complete the test within 2 hours.
  • Once you have completed the test, please send it using wetransfer (as e-mail systems might block php and js files) to steven@aerocrs.com.
